Getting Started

Learn how to integrate Panierr into your project

PanierrCore

import PanierrCore from 'panierr';

const cart = new PanierrCore({
    autoSave: true,
    discounts: [
        { code: 'SAVE10', type: 'percentage', amount: 10 },
        { code: 'FLAT5', type: 'fixed', amount: 5 }
    ],
    currencyFormat: { style: 'currency', currency: 'USD' },
    locale: 'en-US'
});

// Add an item
cart.addItem({
    id: '1',
    name: 'Product 1',
    price: 29.99,
    quantity: 1
});

// Apply a discount
cart.applyDiscount('SAVE10');

// Get totals
const totals = cart.calculateTotals();
console.log(totals.finalPrice); // Final price after discount

PanierrDOM

import PanierrDOM from 'panierr/dom';

const panierrDOM = new PanierrDOM({
    cart: new PanierrCore(),
    currencyFormat: { style: 'currency', currency: 'USD' },
    locale: 'en-US'
});

// Initialize and render the cart
panierrDOM.renderCart();

Basic Usage

Integrate PanierrCore and PanierrDOM to manage your shopping cart seamlessly.

import PanierrCore from 'panierr';
import PanierrDOM from 'panierr/dom';

const cart = new PanierrCore({
    discounts: [{ code: 'SAVE10', type: 'percentage', amount: 10 }]
});

const panierrDOM = new PanierrDOM({
    cart,
    currencyFormat: { style: 'currency', currency: 'USD' },
    locale: 'en-US'
});

// Add an item programmatically
cart.addItem({
    id: '2',
    name: 'Product 2',
    price: 49.99,
    quantity: 2
});

// Render the cart
panierrDOM.renderCart();

Handling Events

Listen to cart events to trigger custom behaviors.

cart.on('itemAdded', (event) => {
    console.log('Item added:', event.item);
});

cart.on('discountApplied', (event) => {
    console.log('Discount applied:', event.discount);
});
